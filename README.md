# Dict
波波字典

功能|简介|备注
--|--|--
汉字拼音查询|通过拼音查询汉字|可以查看汉字 部首、笔画、五笔 基本释义、详细解释（部分文字无）
汉字部首查询|通过部首查询汉字|可以查看汉字 部首、笔画、五笔 基本释义、详细解释（部分文字无）
四字成语查询|输入一个成语查看解释|同义词（个别成语无）、反义词（个别成语无）、出处、例句、语法、引证、英汉互译。
图文识别|通过相机或本地相册|识别出上面的文字
